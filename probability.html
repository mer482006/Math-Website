PROBABILITY


Key Points

  • All outcomes of different experiments are not equally likely.
  • An experiment having one outcome of an event is called an elementary event.
  • Probability of an event = Number of favourable outcomes / Total number of outcomes
  • Sum of all probabilities of an elementary event is 1.
  • P (Ē) = 1 - P (E)
  • Ē is the complement of E. Hence, E and Ē are complementary events.
  • When P(E) = 0, it is an impossible event.
  • When P(E) = 1, it is a possible event.
  • 0 ≤ P(E) ≤ 1
